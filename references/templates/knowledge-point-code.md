## 九、编程三层检查清单

学完本知识点后，确认你能做到：

- [ ] **现象层**：不看解释，能预测一段新代码的运行结果
- [ ] **语义层**：能逐行说出每行代码在做什么，每个术语是什么意思
- [ ] **概念层**：能用自己的话描述"什么样的代码属于 [概念名]"，能判断任意新代码是否属于此概念
- [ ] **三层打通**：给定一个新需求 → 判断属于什么概念 → 写出代码 → 预测输出 → 运行验证

> 四个都打勾，才算真正建构了这个编程知识。
